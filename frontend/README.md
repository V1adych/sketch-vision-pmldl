# Go Frontend Sandbox

Minimal Go equivalents of the previous Java sandbox.

- `internal/util`: `IsBlank`, `Repeat`, `Join`, `LRUCache`
- `internal/model`: generic `Pair[A,B]`
- `internal/algo`: `Matrix` with `Identity`, `From`, `Multiply`
- `cmd/app`: tiny demo main

Run:

```bash
cd frontend
go run ./cmd/app
```
